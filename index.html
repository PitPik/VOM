This demo shows (a) button(s) with a text that represents the number of clicks being fired on itself.

In case there was no button before this request of the page, it starts with a single button that was clicked 0 times and can be clicked 10 times before it would disappear again.

More buttons can be added by adding items to the model in the browser's console. 

demo.appendChild({
    button: true,
    clicks: 8,
    maxClicks: 12
});

The state of an existing button can be manipulated in the browser's console.

demo.model[0].clicks = 8;
although, if the value is smaller or as big as the current value, an error will be shown in the console of the browser.

The states of the buttons are stored in the local storage and are restored on a new page request (refresh page).
